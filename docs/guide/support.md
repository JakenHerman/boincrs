# Support

If `boincrs` is useful to you and you want to support ongoing development,
thank you.

## Financial support

This project currently supports direct funding through maintainer-managed
channels.

- [GitHub Sponsors](https://github.com/sponsors/JakenHerman)
- [Buy Me a Coffee](https://buymeacoffee.com/jakenherman)

## Other ways to help

- **Report bugs** with reproducible steps and a diagnostics bundle (`D` in the
  TUI). See [Reporting bugs](./contributing.md#reporting-bugs).
- Contribute docs and onboarding improvements to the
  [docs site](https://jakenherman.github.io/boincrs).
- Submit UI / UX improvements for terminal readability.
- Help test on Linux, macOS, and Windows — see
  [Compatibility](./compatibility.md).
- Share `boincrs` with BOINC and open science communities.

## Funding priorities

1. Stability and bug fixes.
2. Cross-platform packaging and release automation.
3. Documentation and onboarding.
4. Feature development and long-term maintenance.

## Security reports

Do not post BOINC passwords, project account keys, or full `.env` contents
in public issues. For security-sensitive reports, contact the maintainer
directly through a channel listed on their GitHub profile.
